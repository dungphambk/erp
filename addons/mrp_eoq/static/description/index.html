MRP Economic Order Quantity / Production Order Quantity / Discount Quantity Model

This module helps to calculate the economic order quantity, production order quantity and discount quantity model.

Requirements
  • This moudle require the base MRP module.
  • The Sale and Product moudle is also required to work properly.
Basic Features
  • Calculate the economic order quantity from existed data and input.
  • Calculate the production order quantity from existed data and input.
  • Apply the discount quantity model from existed data and input.
  • Provide related information when making a purchase order.
